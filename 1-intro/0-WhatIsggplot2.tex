	\noindent\textbf{About ggplot}

	
\textbf{ggplot} is a graphics package for Python that aims to approximate R's ggplot2 package in both usage and aesthetics.\\
\bigskip
\textbf{Authors:} Greg Lamp and Austin Ogilvie
